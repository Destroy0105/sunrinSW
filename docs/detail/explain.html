스마트폰을 활용하여 스마트폰 게임 자동 플레이

프로젝트 설명

이 프로젝트는 아두이노를 이용하여 게임 '무한의 계단'을 자동으로 플레이하는 장치를 만드는 것입니다!

원리

RGB 컬러센서 (TCS3200)를 통해 게임 화면 중 필요한 부분(다음 발판이 오는 위치)을 인식합니다.
인식한 데이터를 바탕으로 다음 발판이 왼쪽, 오른쪽 중 어디에 있는지 알아내고
그에 알맞게 서보모터를 움직여 화면을 터치함으로써 게임을 자동으로 플레이합니다.

부품 설명

아두이노 Uno

여러가지 센서와 전자 부품들을 연결하여 제어할 수 있는 장치로 소형 컴퓨터라고 생각하면 됩니다.

RGB컬러센서(TCS3200)

색을 인식하는 센서입니다.

서보모터

원하는 위치나 원하는 각도만큼 회전시킬 수 있는 모터입니다.
